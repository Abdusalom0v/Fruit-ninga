
🥷 Finger Ninja: AI-Powered Web Game
Finger Ninja is an interactive, web-based reimagining of the classic fruit-slicing genre. Instead of using a mouse or a touchscreen, this game leverages Artificial Intelligence and Computer Vision to turn your index finger into a virtual blade.

✨ Key Features
AI-Powered Hand Tracking: Uses the MediaPipe Hands framework to detect and track your finger movements in real-time.

Touchless Interface: Play the game entirely through hand gestures via your webcam—no physical contact with the screen required.

Real-time Game Physics: Features realistic gravity, fruit rotation, and dynamic particle effects for sliced objects.

Dynamic Sound Effects: Includes procedurally generated audio for slicing actions and bomb explosions using the Web Audio API.

Interactive UI: Complete with score tracking, a 60-second countdown timer, and a health (lives) system.

🛠 Built With
HTML5 & CSS3: For the game structure and responsive visual styling.

Vanilla JavaScript: Handles the core game logic, physics engine, and collision detection.

MediaPipe Hands: Google's high-fidelity hand and finger tracking solution.

HTML5 Canvas API: Used for rendering all game animations and the finger "slice" trail.

🚀 How to Get Started
Clone or Download: Save the index.html file to your local machine.

Open in Browser: Open the file using any modern web browser (Chrome, Edge, or Firefox).

Grant Permissions: Allow the browser to access your webcam when prompted.

Play: Click the "НАЧАТЬ ИГРУ" (Start Game) button and start slicing fruits with your index finger!

🎮 Gameplay Rules
Fruits: Slice different fruits (Apple, Orange, Banana, Watermelon) to gain points.

Bombs: Avoid the bombs! Slicing a bomb or letting too many fruits fall will cost you a life.

Goal: Try to get the highest score possible before the 60-second timer runs out or you lose all 3 lives.
